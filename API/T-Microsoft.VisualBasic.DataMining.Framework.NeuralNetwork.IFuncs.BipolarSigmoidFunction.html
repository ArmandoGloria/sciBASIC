BipolarSigmoidFunction

namespace: Microsoft.VisualBasic.DataMining.Framework.NeuralNetwork.IFuncs

Bipolar sigmoid activation function.

The class represents bipolar sigmoid activation function with
the next expression:

2

f(x) = —————— - 1
1 + exp(-alpha * x)

2 * alpha * exp(-alpha * x )

f’(x) = ——————————– = alpha (1 - f(x)^2) / 2
(1 + exp(-alpha x))^2

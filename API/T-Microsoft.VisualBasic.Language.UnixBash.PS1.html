PS1

namespace: Microsoft.VisualBasic.Language.UnixBash

PS (Prompt Sign)

Methods

#ctor

Parameter Name Remarks
ps1 -

Example as:

A

\A 显示时间, 24小时格式: HH:MM

d

\d 代表日期, 格式为Weekday Month Date, 例如Mon Aug 1

Fedora12

Fedora 12 PS1 variable of the bash shell: [\u@\h \w \A #\#]\$

n

\# 下达的第几个指令。

r

\$ 提示字符，如果是root时，提示字符为#，否则就是$。

T

\T 显示时间, 12 小时的时间格式!

tl

\t 显示时间, 为 24 小时格式, 如: HH:MM:SS

u

\u 目前使用者的账号名称

v

\v BASH的版本信息

W

\W 利用@"M:Microsoft.VisualBasic.ProgramPathSearchTool.BaseName(System.String)"取得工作目录名称，所以仅会列出最后一个目录名。

wl

\w 完整的工作目录名称。家目录会以~取代

Properties

H

\H 完整的主机名称

hl

\h 仅取主机名称的第一个名字

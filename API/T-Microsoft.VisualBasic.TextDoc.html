IsTextFile

Determined that the target file is a text file or binary file?
(判断是否是文本文件)

Parameter Name Remarks
FilePath 文件全路径名称
chunkSize 文件检查的长度，假若在这个长度内都没有超过null的阈值数，则认为该文件为文本文件，默认区域长度为4KB

returns: 是返回True，不是返回False

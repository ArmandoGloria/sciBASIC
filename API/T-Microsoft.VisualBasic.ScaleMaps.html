GenerateMapping

如果每一个数值之间都是相同的大小，则返回原始数据，因为最大值与最小值的差为0，无法进行映射的创建（会出现除0的错误）

data -

Scale

TrimRanges

Trims the data ranges,
if n in Dbl vector is less than min, then set n = min;
else if n is greater than max, then set n value to max,
else do nothing.

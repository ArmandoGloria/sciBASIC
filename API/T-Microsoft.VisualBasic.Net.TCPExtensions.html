GetFirstAvailablePort

Get the first available TCP port on this local machine.
(获取第一个可用的端口号，请注意，在高并发状态下可能会出现端口被占用的情况，
所以这时候建议将BEGIN_PORT设置为-1，则本函数将会尝试使用随机数来分配可用端口，从而避免一些系统崩溃的情况产生)

Parameter Name Remarks

LQuerySchedule

namespace: Microsoft.VisualBasic.Parallel.Linq

Parallel Linq query library for VisualBasic.
(用于高效率执行批量查询操作和用于检测操作超时的工具对象，请注意，为了提高查询的工作效率，请尽量避免在查询操作之中生成新的临时对象
并行版本的LINQ查询和原始的线程操作相比具有一些性能上面的局限性)
